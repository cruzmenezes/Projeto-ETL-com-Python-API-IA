# 🚀 Projeto ETL com Python + API + IA

## 📌 Descrição

Este projeto demonstra a construção de um pipeline ETL (Extração, Transformação e Carregamento), integrando dados de uma planilha, consumo de API e uso de Inteligência Artificial para geração de mensagens personalizadas.

---

## 🧠 Conceito do ETL

- **Extração:** Ler dados de uma planilha e buscar informações em uma API
- **Transformação:** Utilizar IA para gerar mensagens personalizadas
- **Carregamento:** Enviar os dados transformados de volta para a API

---

## 🛠️ Tecnologias utilizadas

- Python
- Pandas
- Requests
- OpenAI API
